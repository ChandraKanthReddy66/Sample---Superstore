# Retail Store Sales Analytics Dashboard

## Project Overview

The Retail Store Sales Analytics Dashboard is an end-to-end Business Intelligence project developed using Power BI and MySQL. This project analyzes retail sales transactions to provide actionable insights into business performance, customer behavior, and product profitability.

The dashboard enables stakeholders to monitor KPIs, identify trends, evaluate customer segments, and make data-driven decisions.

## Project Objectives

* Analyze overall retail sales performance.
* Identify top-performing regions, customers, and products.
* Evaluate customer purchasing behavior.
* Measure product profitability and category contribution.
* Perform Year-over-Year (YoY) sales analysis.
* Provide interactive business insights through dashboards.

## Tools and Technologies

* Power BI Desktop
* MySQL
* DAX (Data Analysis Expressions)
* Power Query
* Data Modeling
